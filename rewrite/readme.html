eAthena SVN series

eAthena is an open-source Ragnarok Online server emulator. It's written in C, but we are working on a C++ version. Although it is cross-platform, we only officially support Win32 and Linux.

eAthena is licensed under the GPL, so please give us credit if you use our code.
Our SVN (You may download TortoiseSVN here to access our SVN) is located at http://66.118.142.23:8080/svn/ea/. SVN stands for Subversion, which is similar to the commonly used CVS.

P.S. If you had a hard time loading this readme, please use Firefox.
